# Networking Fundamentals

This directory documents my networking learning journey and covers
fundamental networking concepts required for cybersecurity and SOC analysis.

## Topics Covered

1. Networking Fundamentals
2. OSI and TCP/IP Models
3. IP Addressing and Subnetting
4. Ethernet, MAC Addresses, and ARP
5. Routing, TTL, ICMP, and NAT
6. TCP, UDP, and Ports
7. DHCP and DNS
8. HTTP and Web Communication
9. File Transfer Protocols
10. Email Protocols
11. TLS, SSH, and Secure Communication
12. Network Security Fundamentals

## Learning Objective

The goal of these notes is to build a strong understanding of how devices
communicate across networks and how network traffic can be analyzed from
a cybersecurity and SOC perspective.
